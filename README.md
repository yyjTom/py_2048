此为2048小游戏的python实现版，在此中，我们也能对python的一些小知识有一些认识。

文件夹中curses-2.2-cp35-none-win32.whl，为在python3.5中运行curses包中_curses中缺失文件，在http://www.lfd.uci.edu/~gohlke/pythonlibs/，网站进行下载，并通过pip install *.whl进行安装

指令：

Ord（char）――》获取字符的ascii码

Chr（int）――》输入ascii码返回字符

zip()：zip([seql, ...])接受一系列可迭代对象作为参数，将对象中对应的元素打包成一个个tuple（元组），然后返回由这些tuples组成的list（列表）。若传入参数的长度不等，则返回list的长度和参数中长度最短的对象相同

Defaultdict（Object） 创建一个可以没有key也不报错的字典,且限定了字典的默认值为Object

Lambda：lambda函数也叫匿名函数，即，函数没有具体的名称,而用def创建的方法是有名称的
